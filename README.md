# SFCスマホビューアー

SXF（SFC）図面をスマートフォンで閲覧・計測できるWebツール

👉 https://iku190t.github.io/ais-survey-tool/

---

## ■できること
- SFC図面の表示
- GPSで現在地表示（目安）
- 距離測定
- 文字検索
- メモ機能

---

## ■対象
測量 / 調査 / 設計 / 建設工事

---

## ■特徴
- インストール不要（ブラウザで使用）
- スマホ対応
- 現場で使いやすい設計

---

## ■使い方
以下のURLからそのまま利用できます
